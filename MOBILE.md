# 📱 Guide d'installation mobile

## Installation sur l'écran d'accueil

### 🍎 iPhone / iPad (iOS)

1. **Ouvrir Safari**
   - Allez sur: https://eg-simulator.streamlit.app

2. **Ajouter à l'écran d'accueil**
   - Appuyez sur l'icône "Partager" (↗️)
   - Scroll down, sélectionnez "Sur l'écran d'accueil"
   - Appuyez sur "Ajouter"

3. **C'est fait!** 🎉
   - L'app s'affichera comme une vraie app native
   - Fonctionne hors ligne (grâce au Service Worker)
   - Pas de barre Safari

### 🤖 Android

#### Méthode 1: Chrome (Recommandé)

1. **Ouvrir Chrome**
   - Allez sur: https://eg-simulator.streamlit.app

2. **Ajouter à l'écran d'accueil**
   - Appuyez sur les 3 points (⋮)
   - Sélectionnez "Installer l'app" ou "Ajouter à l'écran d'accueil"
   - Appuyez sur "Installer"

3. **C'est fait!** 🎉

#### Méthode 2: Firefox

1. **Ouvrir Firefox**
   - Allez sur: https://eg-simulator.streamlit.app

2. **Ajouter à l'accueil**
   - Appuyez sur les 3 lignes (☰)
   - Sélectionnez "Ajouter à l'écran d'accueil"
   - Confirmez

### 📲 Edge (Windows, Android)

1. **Ouvrir Edge**
   - Allez sur: https://eg-simulator.streamlit.app

2. **Installer**
   - Appuyez sur l'icône "Installer" (↙️) en haut à droite
   - Confirmez

## 🔧 Troubleshooting

### "L'app ne s'affiche pas"
→ Vérifiez votre connexion internet  
→ Essayez de **recharger** (pull down pour refresh)  
→ Effacez le cache du navigateur

### "Pas de bouton installer"
→ L'app s'installe automatiquement après quelques visites  
→ Essayez manuellement via chrome (⋮ → Ajouter à l'écran)

### "Fonctionne lentement"
→ Vérifiez votre connexion  
→ Videz le cache de l'app  
→ Redémarrez le téléphone
